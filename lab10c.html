Problem 3:

Installation Steps:
  1. First, I selected the Materialize CSS and JS framework for this problem, specifically version 2.1.1 which is the latest version.
  2. To install the Materialize CSS framework, I imported the open-source Content Delivery networks.
  3. In the <head> section of my code I used the CSS link: https://cdn.jsdelivr.net/npm/@materializecss/materialize@2.1.1/dist/css/materialize.min.css
  4. In the <body> section of my code, I used the JavaScript link: https://cdn.jsdelivr.net/npm/@materializecss/materialize@2.1.1/dist/js/materialize.min.js
Steps to build the site:
  1. Using the framework, I created <div> elements containing the "container" classes to create one container for my title, and the other containing my paragraphs detailing my steps. The containers use the `.container` class from Materialize CSS, automatically applying margins and width.
  2. I then created <div> elements that contain “card-content” and “card” classes to contain the <p> paragraph elements with my write-ups. These card classes are also derived from the Materialize framework.
  3. The "row" and "col s12" classes ensure the cards stretch across the whole container. These too are from the Materialize framework.
  4. Lastly, I used the <ol> and <li> elements, which are ordered lists that use the automatic styling from Materialize.
Challenged I faced, and solutions to overcome:
  1. The first challenge I faced was understanding the grid system that Materialize uses. For this I had to watch YouTube videos on the Materialize framework to understand how to properly utilize the tool.
  2. The second issue I faced was hosting the website on DigitalOcean. I opted for creating a GitHub repository containing this HTML file that DigitalOcean then pulls from.
